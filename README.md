# Project Overview

Using the song "Slipping Through My Fingers" by ABBA, I developed a portion of the song into an animated lyric website. By exploring a lot of nostalgic feelings with the use of motion, I attempted to show the internal monolgue of a mother struggling with her daughter growing up. 

# Technical Overview

This project was created by coding in both CSS and HTML. It used a total of 5 pages, including a landing page, and one large CSS file. Animation techniques such as background, font shadows, and movement of text were all applied to show motion and sizing changes. 

Colors were applied intentionally as ABBA has always used specific colors for the album this song is featured on, primarily blues, yellows, and golds.
